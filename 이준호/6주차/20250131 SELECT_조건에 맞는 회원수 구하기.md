## 문제

USER_INFO 테이블에서 2021년에 가입한 회원 중 나이가 20세 이상 29세 이하인 회원이 몇 명인지 출력하는 SQL문을 작성해주세요.

## 문제 풀이

1. LIKE 를 사용하여 2021년에 가입한 회원들만 찾기
2. 이후 AGE 가 20 이상 29 이하인 회원을 BETWEEN 을 사용하여 출력 
## SQL 코드

```sql
SELECT
    COUNT(*)
FROM
    USER_INFO
WHERE
    JOINED LIKE "2021%" 
    AND
    AGE BETWEEN 20 AND 29;
```
