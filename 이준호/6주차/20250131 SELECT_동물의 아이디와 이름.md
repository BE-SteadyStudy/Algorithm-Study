## 문제

동물 보호소에 들어온 모든 동물의 아이디와 이름을 ANIMAL_ID순으로 조회하는 SQL문을 작성해주세요.

## 문제 풀이

1. 동물 ID를 기준으로 정렬하여 결과 출력

## SQL 코드

```sql
SELECT
    ANIMAL_ID,
    NAME
FROM
    ANIMAL_INS
ORDER BY
    ANIMAL_ID ASC;
    
```
