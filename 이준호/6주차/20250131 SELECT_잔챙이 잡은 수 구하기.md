## 문제

잡은 물고기 중 길이가 10cm 이하인 물고기의 수를 출력하는 SQL 문을 작성해주세요.</br>
물고기의 수를 나타내는 컬럼 명은 FISH_COUNT로 해주세요.</br>
단, 잡은 물고기의 길이가 10cm 이하일 경우에는 LENGTH 가 NULL 이며, LENGTH 에 NULL 만 있는 경우는 없습니다.

## 문제 풀이

1. LENGTH 가 NULL 인 경우만 찾아서 출력

## SQL 코드

```sql
SELECT
    COUNT(*) AS FISH_COUNT
FROM
    FISH_INFO
WHERE
    LENGTH IS NULL;
```
