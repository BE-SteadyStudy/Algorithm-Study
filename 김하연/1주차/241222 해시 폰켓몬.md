몇 종류가 있는 지만 알면 될 것 같아 Set으로 풀었다.

```java
import java.util.*;

class Solution {
    public int solution(int[] nums) {
        Set<Integer> type = new HashSet<>();

        for (int i : nums) {
            type.add(i);
        }

        return Math.min(type.size(), nums.length/2);
    }
}
```
